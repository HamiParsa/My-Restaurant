## 📝 About the Project

**MyRestaurant** is a modern restaurant website built with **React, Next.js, TypeScript, Tailwind CSS, Framer Motion, and React Icons**. Users can explore the home page with hero slider, menu page with search/filter, reservation forms, and About page with interactive features.

---

## ✨ Features

<ul>
  <li>Responsive Navbar with active link highlight and hamburger menu for mobile</li>
  <li>Hero slider on Home page with thumbnails and smooth animations</li>
  <li>Menu page with search box, category filters, and animated dish cards</li>
  <li>Reservation form with date & time picker, icons, and hover effects</li>
  <li>About page with feature cards, icons, and CTA button</li>
  <li>Footer with social icons and professional styling</li>
  <li>Responsive design for mobile, tablet, and desktop</li>
  <li>Modern UI with shadows, gradients, rounded cards, and hover effects</li>
</ul>

## 💡 Description

**MyRestaurant** lets users:

- Explore home page with clickable hero slider thumbnails  
- Browse menu items with search and category filters  
- Fill reservation forms with date & time picker  
- Learn about restaurant features in interactive About page  
- Experience responsive layout with smooth Framer Motion animations  

The project demonstrates:

- Advanced front-end development with React, Next.js, TypeScript  
- Modern UI/UX design principles: gradients, shadows, hover effects  
- Responsive and interactive experience suitable for portfolios
